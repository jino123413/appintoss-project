# 편혜 UX 흐름 캡처 (Playwright)

## 캡처 시나리오
1. 홈(목록) 기본 상태
2. 정렬 `가격낮은순` 적용
3. 비교 탭 진입
4. 비교 상세 확장
5. 북마크 탭 진입
6. 북마크 갱신 체크 결과

## 실행 커맨드
백엔드 루트(`pyeonhye-backend`)에서 실행:

```bash
npm run ux:capture
```
