# 편혜 스토리라인

## Core Narrative
1. 발견: 지금 필요한 행사상품을 빠르게 찾는다.
2. 비교: 같은 상품을 브랜드별로 비교한다.
3. 저장: 다음 구매를 위해 북마크에 보관한다.
4. 재확인: 북마크 갱신 체크로 변동을 확인한다.

## UX 톤
- 빠름: 1초 안에 첫 결과가 보이는 구조
- 명확함: 행동 단위 버튼 라벨
- 절제됨: 설명보다 데이터 중심

## 광고 내러티브
- 핵심 탐색은 무료
- 확장 행동(비교 상세, 북마크 갱신 체크)에서만 AD 노출
- 사용자에게 광고 가치가 분명한 지점에서만 호출
