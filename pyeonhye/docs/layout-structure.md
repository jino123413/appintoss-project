# 편혜 레이아웃 구조 (미니멀 UX)

## 목표
- 핵심 행동(검색/필터/탐색)에만 집중
- 설명 문구/배지 남발 제거
- 광고는 부가 행동 버튼에서만 노출

## 화면 구조

```text
[HOME]
 ├─ Header: 앱명 + 새로고침 아이콘
 ├─ Search: 상품명 검색
 ├─ Brand Chips: ALL/CU/GS25/SEVEN/EMART24
 ├─ Promo Chips: ALL/1+1/2+1/DISCOUNT
 └─ Tabs
    ├─ 목록: 카드 스캔 + 북마크
    ├─ 비교: 동일 상품 그룹 비교 (상세 열기 AD)
    └─ 북마크: 저장 상품 + 갱신 체크(AD)
```

## ASCII 와이어

```text
+------------------------------------------+
| 편혜                              [↻]     |
+------------------------------------------+
| [🔍] 상품명 검색                         |
+------------------------------------------+
| [ALL][CU][GS25][SEVEN][EMART24]          |
| [ALL][1+1][2+1][DISCOUNT]                |
+------------------------------------------+
| [목록][비교][북마크]                      |
+------------------------------------------+
| 브랜드 / 상품명                           |
| [1+1] [카테고리]                          |
| 행사가 / 정가                             |
|                                    [★]    |
+------------------------------------------+
```

## 카피 가이드
- 유지: 버튼 라벨, 상태/오류 메시지, 가격 정보
- 제거: 탭 설명 배너, 중복 안내문, 장식성 배지
- 광고 라벨: 부가 행동 버튼의 작은 `AD` 배지로만 제한
