# 오늘의 명대사

매일 하나의 명대사를 보여주고, 좋아요한 문장을 `내 명대사북`에 저장하는 앱입니다.

## 핵심 정책
- 공유 기능 미구현 (공유카드/SNS 연동 없음)
- 핵심 기능 무료
- 부가 기능(다시하기/세부 기능/추가 콘텐츠)만 광고 게이팅

## 개발
```bash
npm install --legacy-peer-deps
npm run dev
npm run build
```
