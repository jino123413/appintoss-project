# 오늘의 명대사 — 레이아웃 구조도

## 홈 화면 (today)

```
┌───────────────────────────────────┐
│ Header: 오늘의 명대사 + 연속 n일    │
├───────────────────────────────────┤
│ Tab: 오늘 문장 | 명대사북 | 연속 기록 │
├───────────────────────────────────┤
│ [카테고리 칩]        [좋아요 저장]    │
│ "오늘의 명대사 본문"               │
│ 출처: 영화/드라마/책/인물            │
├───────────────────────────────────┤
│ [AD] 다시하기 버튼                  │
│ 광고 시청 후 다른 문장 노출          │
├───────────────────────────────────┤
│ 세부 기능 카드                       │
│ - 잠금 상태: [AD] 세부 기능 보기      │
│ - 해금 상태: 확장 해석 노출           │
├───────────────────────────────────┤
│ 히든 명대사 카드                     │
│ - 30일 미달: [AD] 추가 콘텐츠 열기    │
│ - 30일 달성: 히든 명대사 무료 해금    │
└───────────────────────────────────┘
```

## 명대사북 화면 (book)

```
┌───────────────────────────────────┐
│ 카테고리 필터: 전체/사랑/용기/위로/유머│
├───────────────────────────────────┤
│ 저장된 문장 리스트                  │
│ - 문장 본문                         │
│ - 카테고리/출처 메타                │
└───────────────────────────────────┘
```

## 연속 기록 화면 (streak)

```
┌───────────────────────────────────┐
│ 현재 연속 일수 / 최고 기록          │
├───────────────────────────────────┤
│ 30일 진행 바 (0~100%)              │
├───────────────────────────────────┤
│ 해금 상태 안내                      │
│ - 미달: n일 더 방문 필요            │
│ - 달성: 히든 명대사 해금 완료       │
└───────────────────────────────────┘
```

## 광고/핵심 기능 분리

- 핵심 기능(무료): 오늘 문장 확인, 좋아요 저장, 명대사북 조회
- 광고 게이팅: 다시하기, 세부 기능 보기, 추가 콘텐츠 열기
- AD 버튼은 배지 + 사전 고지 문구를 항상 노출

## 차별화 포인트

- 일일 콘텐츠 + 개인 컬렉션 + 30일 해금 루프를 단일 흐름으로 통합
- 점수/등급 없이 문장 중심 감성 카드 구조
- 공유 기능 없이 감상/저장/재방문 루프에 집중
